# Study Assistant (Akıllı Çalışma Asistanı)

Konsoldan ders bilgilerini alıp toplam çalışma süresini hesaplayan, konu başına süreyi çıkaran ve çalışma planını hem ekrana hem de `.txt` dosyasına yazan basit bir Python projesi.

## Özellikler
- Ders adı, günlük çalışma süresi ve gün sayısını alır
- Toplam çalışma süresini hesaplar
- Konu sayısını ve konu isimlerini alır
- Konu başına düşen süreyi hesaplar
- Çalışma planını ekrana yazdırır
- Planı dosyaya kaydeder (örn: `study_plan.txt`)

## Kurulum
Python 3.10+ önerilir.

```bash```
git clone https://github.com/cetinutku2007-ctrl/study-assistant.git
cd study-assistant
python study_assistant.py
